# 🚀 Project Management Tool

A full-stack Project Management Tool built using the MERN Stack.  
This application allows users to create projects, manage tasks, update task statuses, and collaborate through comments.

---

## 📌 Features

### 🔐 Authentication
- User Registration
- User Login
- JWT Authentication

### 📁 Project Management
- Create Projects
- View Projects
- Delete Projects

### ✅ Task Management
- Create Tasks
- Update Task Status
- Delete Tasks

### 💬 Comments System
- Add Comments to Tasks
- View Comments

### 🎨 Frontend
- Responsive UI
- Modern Dashboard
- Dynamic Data Fetching

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Axios
- React Router DOM

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas

## ⚙️ Installation

### 1️⃣ Clone Repository

```bash
git clone https://github.com/Vishwanath810/CodeAlpha_ProjectManagementTool.git
```

---

### 2️⃣ Install Frontend Dependencies

```bash
cd client
npm install
```

---

### 3️⃣ Install Backend Dependencies

```bash
cd ../server
npm install
```

---

### 4️⃣ Configure Environment Variables

Create `.env` inside `server` folder:

```env
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

---

### 5️⃣ Run Frontend

```bash
cd client
npm run dev
```

---

### 6️⃣ Run Backend

```bash
cd server
npm run dev
```
